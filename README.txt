
Hijrah Quest — Canva Embed Package
Untuk SDN Pisangcandi 4

File di paket:
- hijrah-quest-canva.html    : file HTML interaktif (embed-able)
- logo_school.png           : logo sekolah (placeholder)
- cert_template.png         : template sertifikat (grafis)
- README.txt                : instruksi ini

Langkah cepat untuk men-embed di Canva (tanpa hosting server sendiri):
1) Upload file 'hijrah-quest-canva.html' ke Netlify Drop atau GitHub Pages:
   - Netlify Drop: https://app.netlify.com/drop  (seret & lepas file .html dan assets ke sana)
   - GitHub Pages: buat repo baru, upload file, aktifkan GitHub Pages (branch main, folder root)

2) Setelah hosting aktif, Anda akan mendapat URL seperti https://your-site.netlify.app/hijrah-quest-canva.html
   Salin URL tersebut.

3) Di Canva: buka desain Anda -> Apps -> Embed -> 'Embed a website' -> paste URL -> pilih 'Embed'
   Atur ukuran frame embed sesuai kebutuhan. Pastikan 'Allow fullscreen' diaktifkan (opsional).

Opsional (lebih sederhana, jika tidak ingin hosting):
- Unggah file HTML ke Google Drive, dapatkan link publik, lalu gunakan layanan pihak ketiga untuk meng-host HTML.
- Atau gunakan layanan hosting gratis lainnya (Vercel, Surge, Neocities).

Catatan teknis:
- Web Speech API (narasi) bekerja paling baik di Chrome / Edge.
- Ganti file logo_school.png dengan logo asli sekolah jika tersedia.
- File cert_template.png bisa digunakan untuk mengganti background sertifikat di HTML/Canvas.

Jika Bapak mau, saya bisa:
A) Siapkan ZIP berisi file ini (sudah saya buat) — link download tersedia.
B) Upload HTML ke Netlify saya sementara dan berikan URL untuk langsung di-embed (butuh izin).

Silakan gunakan file ZIP atau minta bantuan selanjutnya.
